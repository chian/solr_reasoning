Sequence feature record: Represents a sequence feature and associated metadata in BV-BRC.

A sequence feature record describes a genetic or protein feature, including identifiers, sequence, type, and metadata. Sequence features are used for annotation and variant analysis.

Fields:
- id: Unique identifier for the sequence feature. Example: SF_00001
- feature_id: Feature identifier. Example: fig|83332.12.peg.1
- gene: Gene symbol. Example: dnaN
- genome_id: Identifier of the parent genome. Example: 83332.12
- genome_name: Name of the parent genome. Example: "Escherichia coli K12"
- product: Product name or description. Example: "DNA polymerase III subunit beta"
- aa_sequence_md5: MD5 checksum of the amino acid sequence. Example: 1bc29b36f623ba82aaf6724fd3b16718
- aa_variant: Amino acid variant. Example: "A23T"
- additional_metadata: Additional metadata fields. Example: ["meta1", "meta2"]
- comments: Additional comments. Example: ["predicted variant"]
- date_inserted: Date the record was inserted. Example: 2020-01-01
- date_modified: Date the record was last modified. Example: 2021-01-01
- end: End coordinate of the feature. Example: 2550
- evidence_code: Evidence code for the feature. Example: "IEA"
- genbank_accession: GenBank accession. Example: "U00096.3"
- length: Length of the feature. Example: 350
- patric_id: PATRIC-specific feature ID. Example: fig|83332.12.peg.1
- publication: Publications supporting the feature. Example: ["PMID:12345678"]
- refseq_locus_tag: RefSeq locus tag. Example: "b0001"
- segment: Segment designation. Example: "Segment 1"
- segments: List of segments. Example: ["seg1", "seg2"]
- sf_category: Sequence feature category. Example: "variant"
- sf_id: Sequence feature ID. Example: "SF_00001"
- sf_name: Name of the sequence feature. Example: "DNA polymerase III variant"
- sf_sequence: Sequence of the feature. Example: "ATGCGT..."
- sf_sequence_md5: MD5 checksum of the feature sequence. Example: 1bc29b36f623ba82aaf6724fd3b16718
- source: Source of the feature. Example: "RefSeq"
- source_aa_sequence: Source amino acid sequence. Example: "MSTNPKPQR..."
- source_id: Source-specific ID. Example: "NP_414543.1"
- source_sf_location: Source sequence feature location. Example: "chr1:100-200"
- source_strain: Source strain. Example: "K12"
- start: Start coordinate of the feature. Example: 190
- subtype: Subtype of the feature. Example: "missense"
- taxon_id: NCBI Taxonomy ID. Example: 83332
- variant_types: Types of variants. Example: "SNP"

Notes:
- Sequence features are used for annotation and variant analysis.
- Not all fields are present for every record; see BV-BRC documentation for more details.
