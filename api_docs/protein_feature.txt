Protein feature record: Represents an annotated protein-coding feature within a genome.

A protein feature record describes a protein-coding gene or CDS, including its identifiers, biological function, sequence, and related metadata. Protein features are linked to genomes, protein families, and functional annotations.

Fields:
- id: Unique identifier for the protein feature. Example: P_00001
- feature_id: Unique identifier for the protein feature. Example: fig|83332.12.peg.1
- genome_id: Identifier of the parent genome. Example: 83332.12
- genome_name: Name of the parent genome. Example: Escherichia coli K12
- product: Name or description of the protein's biological function. Example: DNA polymerase III subunit beta
- gene: Gene symbol or name (if available). Example: dnaN
- protein_id: Protein accession (if available). Example: NP_414543.1
- sequence: Amino acid sequence of the protein (if requested).
- ec_number: Enzyme Commission number (if applicable). Example: 2.7.7.7
- figfam_id: FIGfam protein family ID (if applicable). Example: FIG00000002
- plfam_id: Genus-specific protein family ID (if applicable). Example: PLF_000001
- pgfam_id: Cross-genus protein family ID (if applicable). Example: PGF_000001
- annotation: Source of the annotation (e.g., PATRIC, RefSeq, GenBank). Example: PATRIC
- aliases: Alternative names or locus tags. Example: b0084, JW0073
- aa_sequence_md5: MD5 checksum of the amino acid sequence. Example: 1bc29b36f623ba82aaf6724fd3b16718
- classification: Classification terms. Example: ["enzyme"]
- comments: Additional comments. Example: ["predicted by RAST"]
- date_inserted: Date the record was inserted. Example: 2020-01-01
- date_modified: Date the record was last modified. Example: 2021-01-01
- description: Description of the protein. Example: "DNA polymerase III subunit beta"
- e_value: E-value for sequence similarity. Example: 1e-50
- end: End coordinate of the feature. Example: 2550
- evidence: Evidence for the annotation. Example: "experiment"
- feature_type: Type of feature. Example: "CDS"
- interpro_description: InterPro description. Example: "DNA polymerase family"
- interpro_id: InterPro ID. Example: "IPR006134"
- length: Length of the protein in amino acids. Example: 350
- patric_id: PATRIC-specific feature ID. Example: fig|83332.12.peg.1
- publication: Publications supporting the annotation. Example: ["PMID:12345678"]
- refseq_locus_tag: RefSeq locus tag. Example: "b0001"
- score: Score for sequence similarity. Example: 200.5
- segments: Segments of the feature. Example: ["segment1", "segment2"]
- source: Source of the protein data. Example: "RefSeq"
- source_id: Source-specific ID. Example: "NP_414543.1"
- start: Start coordinate of the feature. Example: 190
- taxon_id: NCBI Taxonomy ID. Example: 83332

Notes:
- The 'product' field contains the functional name or description of the protein.
- Protein features are a subset of genome features with feature_type = CDS.
- Not all fields are present for every protein; see BV-BRC documentation for more details.
