Specialty gene record: Represents a specialty gene and associated metadata in BV-BRC.

A specialty gene record describes a gene of interest (e.g., AMR, virulence, resistance), including identifiers, function, evidence, and metadata. Specialty genes are used for annotation and comparative genomics.

Fields:
- id: Unique identifier for the specialty gene. Example: SPG_00001
- gene: Gene symbol. Example: blaCTX-M
- function: Function of the gene. Example: "beta-lactamase"
- product: Product name or description. Example: "Beta-lactamase enzyme"
- feature_id: Feature identifier. Example: fig|83332.12.peg.1
- genome_id: Identifier of the parent genome. Example: 83332.12
- genome_name: Name of the parent genome. Example: "Escherichia coli K12"
- organism: Name of the organism. Example: "Escherichia coli"
- classification: List of classification terms. Example: ["AMR"]
- antibiotics: List of antibiotics associated. Example: ["ampicillin"]
- antibiotics_class: Class of antibiotics. Example: "beta-lactam"
- evidence: Evidence for the gene. Example: "experiment"
- e_value: E-value for sequence similarity. Example: "1e-50"
- identity: Percent identity. Example: 99
- query_coverage: Query coverage percentage. Example: 100
- subject_coverage: Subject coverage percentage. Example: 100
- property: Property of the gene. Example: "resistance", "virulence factor"
- property_source: Source of the property. Example: "CARD"
- pmid: List of PubMed IDs. Example: ["12345678"]
- alt_locus_tag: Alternative locus tag. Example: "b0001"
- refseq_locus_tag: RefSeq locus tag. Example: "b0001"
- patric_id: PATRIC-specific feature ID. Example: fig|83332.12.peg.1
- source: Source of the gene data. Example: "RefSeq"
- source_id: Source-specific ID. Example: "NP_414543.1"
- public: Whether the record is public. Example: true
- owner: Owner of the record. Example: "user@bv-brc.org"
- same_genome: Number of same genome hits. Example: 1
- same_genus: Number of same genus hits. Example: 1
- same_species: Number of same species hits. Example: 1
- taxon_id: NCBI Taxonomy ID. Example: 83332
- user_read: Users with read access. Example: ["user1", "user2"]
- user_write: Users with write access. Example: ["user1"]
- date_inserted: Date the record was inserted. Example: 2020-01-01
- date_modified: Date the record was last modified. Example: 2021-01-01
- _version_: Version number of the record. Example: 1

Notes:
- Specialty genes are used for annotation and comparative genomics.
- Not all fields are present for every record; see BV-BRC documentation for more details.
