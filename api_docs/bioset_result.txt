Bioset result record: Represents the result of a bioset analysis or experiment in BV-BRC.

A bioset result record describes the outcome of a bioset experiment, including identifiers, experiment details, gene/protein information, and statistical results. Results are linked to biosets, experiments, and biological entities.

Fields:
- id: Unique identifier for the bioset result. Example: RESULT_00001
- bioset_id: Identifier of the associated bioset. Example: BIOSET_00001
- bioset_name: Name of the bioset. Example: "E. coli stress response"
- bioset_description: Description of the bioset. Example: "Samples exposed to oxidative stress."
- bioset_type: Type of bioset. Example: "RNA-Seq"
- counts: Raw counts or measurement value. Example: 1000
- date_inserted: Date the record was inserted. Example: 2020-01-01
- date_modified: Date the record was last modified. Example: 2021-01-01
- entity_id: Identifier of the biological entity. Example: "b0001"
- entity_name: Name of the entity. Example: "dnaN"
- entity_type: Type of entity. Example: "gene"
- exp_id: Experiment identifier. Example: "EXP_0001"
- exp_name: Name of the experiment. Example: "E. coli RNA-Seq"
- exp_title: Title of the experiment. Example: "Transcriptome analysis of E. coli"
- exp_type: Type of experiment. Example: "RNA-Seq"
- feature_id: Feature identifier. Example: "fig|83332.12.peg.1"
- fpkm: Fragments Per Kilobase Million. Example: 12.5
- gene: Gene symbol. Example: "dnaN"
- gene_id: Gene identifier. Example: "b0001"
- genome_id: List of genome IDs. Example: ["83332.12"]
- locus_tag: Locus tag. Example: "b0001"
- log2_fc: Log2 fold change. Example: 2.5
- organism: Name of the organism. Example: "Escherichia coli"
- other_ids: Other identifiers. Example: ["NP_414543.1"]
- other_value: Other measurement value. Example: 0.5
- p_value: P-value for statistical significance. Example: 0.01
- patric_id: PATRIC-specific feature ID. Example: "fig|83332.12.peg.1"
- product: Product name or description. Example: "DNA polymerase III subunit beta"
- protein_id: Protein identifier. Example: "NP_414543.1"
- result_type: Type of result. Example: "differential expression"
- strain: Strain designation. Example: "K12"
- taxon_id: NCBI Taxonomy ID. Example: 83332
- tpm: Transcripts Per Million. Example: 20.0
- treatment_amount: Amount of treatment. Example: "10 mM"
- treatment_duration: Duration of treatment. Example: "1 hour"
- treatment_name: Name of the treatment. Example: "H2O2"
- treatment_type: Type of treatment. Example: "oxidative"
- uniprot_id: UniProt identifier. Example: "P0A8V2"
- z_score: Z-score for the result. Example: 1.5
- _version_: Version number of the record. Example: 1

Notes:
- Bioset results provide quantitative and statistical outcomes for bioset analyses.
- Not all fields are present for every record; see BV-BRC documentation for more details.
