BV-BRC MCP Documentation Index
==============================

This index provides an overview of the documentation and data types available via the MCP server, as well as a quick reference to BV-BRC API collections and usage tips.

Available Documentation Files:
-----------------------------
- genome.txt: Genome data record: Represents a sequenced, assembled, and annotated genome in BV-BRC.
- genome_feature.txt: Genome feature record: Represents an annotated feature (gene, CDS, RNA, etc.) within a genome.
- genome_amr.txt: Genome AMR record: Represents antimicrobial resistance (AMR) phenotype and metadata for a genome.
- protein_feature.txt: Protein feature record: Represents an annotated protein-coding feature within a genome.
- pathway.txt: Pathway record: Represents a biological pathway annotated in a genome, typically using KEGG or similar standards.

API Query Tips:
---------------
- Use `limit(1)` or a small number to cap result size.
- Use `select(field1,field2)` to limit columns returned.
- Use filters like `eq(field,value)` and combine with `and()` or `or()`.
- Example: `curl -L 'https://www.bv-brc.org/api/genome/?and(eq(genus,Escherichia),eq(species,coli))&limit(1)'`
- See each documentation file for field details and example queries.
