Genome data record: Represents a sequenced, assembled, and annotated genome in BV-BRC.

A genome is the central data type in BV-BRC, linking to all associated features, metadata, and analyses. Each genome record contains identifiers, organism information, sequencing and assembly details, and links to features and annotations.

Fields:
- genome_id: Unique identifier for the genome. Example: 83332.12
- genome_name: Scientific name and strain. Example: Escherichia coli K12
- organism: Organism name. Example: Escherichia coli
- strain: Strain designation. Example: K12
- taxon_id: NCBI Taxonomy ID. Example: 83332
- genome_status: Status of the genome (e.g., Complete, Draft). Example: Complete
- genome_length: Total length of the genome in base pairs. Example: 4641652
- gc_content: GC content percentage. Example: 50.8
- isolation_site: Site of isolation. Example: Human gut
- collection_date: Date of sample collection. Example: 2001-05-12
- sequencing_platform: Platform used for sequencing. Example: Illumina HiSeq
- assembly_method: Method used for assembly. Example: SPAdes v3.13.0
- bioproject_accession: NCBI BioProject accession. Example: PRJNA12345
- biosample_accession: NCBI BioSample accession. Example: SAMN00123456
- refseq_accession: NCBI RefSeq accession. Example: GCF_000005845.2
- genbank_accession: NCBI GenBank accession. Example: U00096.3
- publication: Associated publication or PMID. Example: PMID:12345678
- _version_: Version number of the record. Example: 1
- additional_metadata: Additional metadata fields. Example: ["field1", "field2"]
- altitude: Altitude of sample collection. Example: "100m"
- antimicrobial_resistance: List of AMR phenotypes. Example: ["ampicillin:Resistant"]
- antimicrobial_resistance_evidence: Evidence for AMR. Example: "Phenotype"
- assembly_accession: Assembly accession. Example: GCA_000005845.2
- authors: Authors of the genome publication. Example: "Smith et al."
- biovar: Biovar designation. Example: "biovar 1"
- body_sample_site: Body site of sample. Example: "gut"
- body_sample_subsite: Subsite of sample. Example: "colon"
- cds: Number of coding sequences. Example: 4321
- cds_ratio: Ratio of CDS to genome length. Example: 0.85
- cell_shape: Cell shape. Example: "rod"
- checkm_completeness: CheckM completeness score. Example: 99.5
- checkm_contamination: CheckM contamination score. Example: 0.2
- chromosomes: Number of chromosomes. Example: 1
- clade: Clade designation. Example: "A"
- class: Taxonomic class. Example: "Gammaproteobacteria"
- coarse_consistency: Coarse consistency score. Example: 1.0
- collection_year: Year of collection. Example: 2001
- comments: Comments about the genome. Example: ["high quality"]
- common_name: Common name of the organism. Example: "E. coli"
- completion_date: Date of genome completion. Example: 2001-06-01
- contig_l50: L50 of contigs. Example: 10
- contig_n50: N50 of contigs. Example: 50000
- contigs: Number of contigs. Example: 100
- core_families: Number of core families. Example: 2000
- core_family_ratio: Ratio of core families. Example: 0.95
- culture_collection: Culture collection info. Example: "ATCC 12345"
- date_inserted: Date record was inserted. Example: 2020-01-01
- date_modified: Date record was modified. Example: 2021-01-01
- depth: Depth of sample. Example: "5m"
- disease: Associated diseases. Example: ["diarrhea"]
- family: Taxonomic family. Example: "Enterobacteriaceae"
- fine_consistency: Fine consistency score. Example: 1.0
- genbank_accessions: GenBank accessions. Example: "U00096.3"
- genome_quality: Quality of the genome. Example: "High"
- genome_quality_flags: Quality flags. Example: ["complete"]
- genus: Taxonomic genus. Example: "Escherichia"
- geographic_group: Geographic group. Example: "Europe"
- geographic_location: Geographic location. Example: "USA"
- gram_stain: Gram stain result. Example: "negative"
- habitat: Habitat of the organism. Example: "intestinal"
- host_age: Age of host. Example: "35"
- host_common_name: Common name of host. Example: "human"
- host_gender: Gender of host. Example: "female"
- host_group: Host group. Example: "mammal"
- host_health: Health status of host. Example: "healthy"
- host_name: Name of host. Example: "Homo sapiens"
- host_scientific_name: Scientific name of host. Example: "Homo sapiens"
- hypothetical_cds: Number of hypothetical CDS. Example: 100
- hypothetical_cds_ratio: Ratio of hypothetical CDS. Example: 0.02
- isolation_comments: Comments on isolation. Example: "from hospital patient"
- isolation_country: Country of isolation. Example: "USA"
- isolation_source: Source of isolation. Example: "feces"
- kingdom: Taxonomic kingdom. Example: "Bacteria"
- lab_host: Laboratory host. Example: "E. coli"
- latitude: Latitude of sample. Example: "40.7128N"
- lineage: Taxonomic lineage. Example: "Bacteria; Proteobacteria; ..."
- longitude: Longitude of sample. Example: "74.0060W"
- mlst: MLST type. Example: "ST131"
- motility: Motility status. Example: "motile"
- ncbi_project_id: NCBI project ID. Example: "PRJNA12345"
- optimal_temperature: Optimal growth temperature. Example: "37C"
- order: Taxonomic order. Example: "Enterobacterales"
- organism_name: Organism name. Example: "Escherichia coli"
- other_clinical: Other clinical metadata. Example: ["ICU"]
- other_environmental: Other environmental metadata. Example: ["soil"]
- other_names: Other names for the organism. Example: ["coli"]
- other_typing: Other typing information. Example: ["serotype O157:H7"]
- owner: Owner of the genome record. Example: "user@bv-brc.org"
- oxygen_requirement: Oxygen requirement. Example: "facultative anaerobe"
- passage: Passage history. Example: "passaged 3 times"
- pathovar: Pathovar designation. Example: "O157:H7"
- patric_cds: Number of PATRIC CDS. Example: 4300
- phenotype: Phenotypes. Example: ["resistant to ampicillin"]
- phylum: Taxonomic phylum. Example: "Proteobacteria"
- plasmids: Number of plasmids. Example: 2
- public: Whether the genome is public. Example: true
- reference_genome: Reference genome status. Example: "yes"
- refseq_accessions: RefSeq accessions. Example: "GCF_000005845.2"
- refseq_cds: Number of RefSeq CDS. Example: 4200
- rrna: Number of rRNA genes. Example: 22
- salinity: Salinity of environment. Example: "0.9%"
- sequencing_centers: Sequencing centers. Example: "Broad Institute"
- sequencing_depth: Sequencing depth. Example: "100x"
- sequencing_status: Sequencing status. Example: "complete"
- serovar: Serovar designation. Example: "O157"
- species: Taxonomic species. Example: "Escherichia coli"
- sporulation: Sporulation status. Example: "non-sporulating"
- sra_accession: SRA accession. Example: "SRR123456"
- state_province: State or province. Example: "New York"
- subclade: Subclade designation. Example: "A1"
- subtype: Subtype designation. Example: "H7"
- superkingdom: Superkingdom. Example: "Bacteria"
- taxon_lineage_ids: Taxonomic lineage IDs. Example: [83332, 562]
- taxon_lineage_names: Taxonomic lineage names. Example: ["Bacteria", "Proteobacteria"]
- temperature_range: Temperature range. Example: "mesophilic"
- trna: Number of tRNA genes. Example: 86
- type_strain: Type strain status. Example: "yes"
- user_read: Users with read access. Example: ["user1", "user2"]
- user_write: Users with write access. Example: ["user1"]
- h1_clade_global: H1 clade global classification. Example: ["clade1", "clade2"]
- h1_clade_us: H1 clade US classification. Example: ["cladeA", "cladeB"]
- h3_clade: H3 clade classification. Example: ["cladeX"]
- h5_clade: H5 clade classification. Example: ["cladeY"]
- mat_peptide: Number of mature peptides. Example: 5
- missing_core_family_ids: List of missing core family IDs. Example: ["PLF_000001"]
- n_type: N type classification. Example: 1
- nearest_genomes: List of nearest genome IDs. Example: ["83332.13", "83332.14"]
- outgroup_genomes: List of outgroup genome IDs. Example: ["83332.15"]
- p2_genome_id: Secondary genome ID. Example: 123456
- partial_cds: Number of partial CDS. Example: 10
- partial_cds_ratio: Ratio of partial CDS. Example: 0.01
- ph1n1_like: PH1N1-like classification. Example: "yes"
- plfam_cds: Number of PLfam CDS. Example: 4200
- plfam_cds_ratio: Ratio of PLfam CDS. Example: 0.85
- refseq_project_id: RefSeq project ID. Example: "PRJNA54321"
- season: Season of sample collection. Example: "spring"
- segment: Segment designation. Example: "Segment 1"
- segments: Number of segments. Example: 8

Notes:
- Many additional metadata fields are available; see BV-BRC documentation for full details.
- All fields may not be present for every genome.
